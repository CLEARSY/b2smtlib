# B2SMTLIB

B2SMTLIB is a translator from pog to SMT-LIB 2.6.

## Usage

```sh
b2smtlib -i <input.pog> -o <output>
```

## Compilation

To compile the project with CMake:

```sh
cmake -B build
cmake --build build
```
